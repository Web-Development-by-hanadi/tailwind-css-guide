# tailwind-css-guide

An interactive, live Tailwind CSS reference — built for **HSam Academy**'s **Web Development** training track.
Guide by **Hind Samet**, Computer Science Engineer.

Demo: open `index.html`, or visit the page once deployed on GitHub Pages (see below).

## What the site contains

A single page (`index.html`), no build step, that explains Tailwind and then displays **every value actually rendered** — not just listed in a table:

- **Introduction** — what Tailwind CSS is, utility-first vs classic CSS, where it's used
- **1. Spacing scale** — bars measured live (real px values)
- **2. Color palette** — 22 families × 11 shades, hex computed live in the browser
- **3. Font size** — text shown at its real size
- **4. Font weight**
- **5. Border radius**
- **6. Box shadow**
- **7. Responsive breakpoints** — live badge that tracks the window width
- **8. Flexbox** — interactive mini playground (justify / items)
- **9. Grid** — interactive mini playground (columns)
- **10. Position & display**
- **11. Width & height (fractions)** — percentage computed live
- **12. Opacity**
- **13. Z-index**
- **14. Transitions & animations**
- **15. State variants** — hover / focus / active / disabled / odd-even /
